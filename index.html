PeerCast

Peer to peer Coding in real time.
Create an account, enter a new room, and invite your friends to code away in real time.

No plugins. No Pre-installation. Just your browser.


This project explores the power WebRTC for real-time communication (RTC) capabilities via simple JavaScript APIs. With this technology, it also enables users to peers to video chat in-browser without the need for any third-party software packages.


Built using Django and Javascript

Our backend is built on Django. It was used to manage user accounts, team projects, and the database. And JavasScript (with Bootstrap integration) allowed for a clean, elegant design with some responsivity.
